    APP SISTEMA EXPERTO

    La APP Sistema experto es una aplicación desarrollada para estimar las dimensiones de una hornilla para la producción de panela, a partir de ciertos parámetros proporcionados por el usuario.

    El objetivo principal de esta aplicación es facilitar al usuario una herramienta que de forma automática genera una propuesta de valor para la implementación de una hornilla, sin que se requiera de un conocimiento de ingeniería especializado. Además, cuenta con un formulario de contacto con un especialista en el área, el cual solucionara las dudas tenga frente al diseño o generara una nueva propuesta si hay parámetros que no se contemplan en el diseño previo.
